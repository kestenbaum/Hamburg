## Projekt-Setup (Linting & Formatting)

Dieses Projekt ist mit grundlegenden Tools für Code-Qualität konfiguriert:

- **ESLint** → prüft JavaScript auf Fehler und schlechte Praktiken
- **Stylelint** → überprüft SCSS/CSS auf konsistente Schreibweise und Struktur
- **Prettier** → sorgt für ein einheitliches Code-Format
- **EditorConfig** → standardisiert Einrückung und Formatierung im Editor

Ziel ist es, sauberen, konsistenten und leicht wartbaren Code zu schreiben und typische Anfängerfehler früh zu vermeiden.
